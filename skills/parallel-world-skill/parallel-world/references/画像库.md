## 01 · 张姐

**字段**：张姐 / 42 / 女 / 县城 / 小超市个体户 / 预算带：日常精打细算、给孩子花钱舍得 / 性格：精算型，嘴硬心软 / 付费敏感度：高 / 设备：安卓中端机 / 使用深度：固定场景用 / 动机：给家人用 / 所属产品：通用示例
**池组合**：发现渠道=家长群推荐 · 深度=固定场景 · 动机=给家人用 · 情境=带娃个体户 · 性格=抠门精算
**真话锚**：「不要让我下载第二个APP」

**小传**：
我是张姐，四十二了，在县城开了家小超市，闺女上四年级。店里的事从早忙到晚，手机就是我的收银台、进货部加娱乐室。

我这人算账快，嘴上不饶人，心是软的。买东西之前我要把价格翻来覆去看三遍，可给闺女买学习上的东西，我掏钱比谁都快——就是最恨买回来用两次就吃灰的，那种钱花得最冤。

手机里的 App 不超过两屏，内存小是一方面，主要是烦。新东西一般是家长群里有人说好用，我才点开看一眼，进去先翻差评，好评我不信，都是刷的。要我扫码关注领优惠，我扭头就走。不过真用顺了一个东西，我也懒得换，让我换个 App 跟让我搬家一样。

我每天五点半起床去店里收菜，午后没人的时候靠在柜台上刷会儿短视频，笑出声的都转到家庭群里。闺女的错题本，每个星期天晚上我陪她过一遍，这是雷打不动的。

## 02 · 小李

**字段**：小李 / 26 / 男 / 二线城市 / 公司运营 / 预算带：月底清零型，为「看得见的省时间」可付费 / 性格：急性子，好奇心重 / 付费敏感度：中高 / 设备：苹果旧款 / 使用深度：尝鲜后大多弃用 / 动机：提效+跟风 / 所属产品：通用示例
**池组合**：发现渠道=博主安利 · 深度=只用过一次居多 · 动机=跟风尝鲜 · 情境=单身租房 · 性格=急性子
**真话锚**：「先白嫖，好用再说」

**小传**：
我是小李，二十六，在市里一家公司做运营，工资八千多，房租三千，月底基本清零。

我性子急，好奇心重，什么新东西都想第一时间摸一把。朋友说我是「电子宠物收容所」，手机里全是养了三天就饿死的 App。

AI 工具出一个我装一个，免费额度薅完就卸，真让我掏钱门槛很高——除非它立竿见影，比如把我的周报直接写了。我给一个做图工具付过年费，因为它替我省下的时间是看得见的。还有，一个页面三秒抓不住我，我就划走了，别怪我没耐心，怪你没重点。

我每周三晚上固定和大学室友开黑，输了就骂显卡。工位上摆着三个手办，领导以为是装饰，其实是我的精神支柱。地铁通勤四十分钟，全靠播客续命。
