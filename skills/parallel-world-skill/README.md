# 平行世界 · 用户推演 Skill（parallel-world）

在对话里养一群 AI 扮演的「平行世界用户」，让你的文案、定价、页面、点子，在见到真用户之前，先被他们评一遍。

原型是作者在一家大型在线教育公司做的企业级用户仿真系统（公司 AI 比赛第一名，部分场景预测与真实结果对照十次对上八次）。这个 skill 是它的脱敏对话版：不用写代码，AI 自己扮演整个平行世界用户群。你只做三件事——

1. **建画像**：告诉它你的用户是哪几类人（它会带你写成有血有肉的小传）
2. **发方案**：把文案 / 价格 / 页面丢给它，它逐人推演，出带原话的报告
3. **喂结果**：方案上线后把真实结果告诉它，它自己修画像，越用越准

每次推演和校准都会**自动生成一张手机版 H5 单页**（零依赖，浏览器直接打开），投屏、发群、给领导看都拿得出手——效果见 `examples/推演-R-001-示例.html`。

## 安装

把 `parallel-world/` 文件夹放进你的 skills 目录：

- **Claude Code**：项目下 `.claude/skills/parallel-world/`（或全局 `~/.claude/skills/`）
- 其他支持 Agent Skills 的工具同理

装好后直接说「帮我建几个用户画像」或「让平行世界用户看看这个方案」。

## 用之前的一个建议

先拿一个你已经知道结果的旧方案「对答案」，对不上先修画像，对上了再测新的。

## 三条边界

- 它是筛子：帮你把三个方案筛成一个、把明显的坑提前排掉
- 真金白银的决定，筛完请再找一两个真人验证
- 画像越真（尤其那句真人原话），推演越准
